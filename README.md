# Spend

a simple to use income and expense tracker

## Technologies

- Next.js
- Mongodb
- Mongoose

## Getting started

### Prerequisites

- Nodejs v22.^
- Mongodb
- npm

### local setup

- clone the repo

  ```bash
    git clone https://github.com/shadinmd/spend && cd spend
  ```

- install dependencies

  ```bash
    npm install
  ```

- setup .env

  ```bash
    cat .env.example > .env
  ```

  edit .env file to set environement variables

- start the dev server
  ```bash
    npm run dev
  ```
